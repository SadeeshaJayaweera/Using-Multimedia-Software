Software Used

In the development of the multimedia product, I have used: various softwares for different tasks and among them, some are online software.

All the software used both online and offline are listed below.

  • Adobe Flash Professional CC 2015

 In the development of the multimedia product and both prototypes, I have used: this software and it allowed me to make a successful multimedia product and also helped to satisfied the customer requirements too by allowing to add many multimedia components.

Once I completed the project this software enabled me to export it In SWF file format. In my opinion, this software is the best tool to develop a multimedia product since it easy to use.

 This software was best in downloading videos from different youtube channels. This software enabled me to download videos from various youtube channels in different resolutions. Thereby i downloaded: all the videos with the use of this software.

 This software was used mostly to cut and edit videos because it has many features to edit videos the main reason that I used this software is? because it was easy and anyone can learn it within few seconds.

 This software did tremendous work by enabled me to type and edit word documents such as sitemaps, etc. And also I used it to convert word files into the pdf file format. However, it helped me to recorrect grammar mistakes too.

 Photoshop was very efficient because it has every single feature that is required to edit pictures and other necessary file formats. I was able to use this software very well since the knowledge I gained in my previous coursework units. Thereby it enabled me to make and edit pictures as well as convert them into different picture formats.
